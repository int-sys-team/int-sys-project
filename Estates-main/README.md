## 📦 Stack
* MongoDB
* Asp.net WebApi
* React (JS)

## 🚀 Quickstart

```bash
> # Clone repository
> git clone https://github.com/djoleant/Estates.git
> cd Estates

> # Server start
> cd EstatesAPI
> dotnet watch run

> # Client start
> cd ../EstatesApp
> npm install
> npm start

> # Start mongoDB
> mongo
> use EstatesDB
```


## ⚙️ Configuration options

| Environment variable name | Default value | 
| ------------------------- | ------------- |
| PORT | 27017 | 
| DB NAME | EstatesDB |
| MongoDB VERSION | 6.0 |
